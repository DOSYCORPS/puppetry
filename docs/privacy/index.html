Information Collection and Use

Puppetry does not collect any personal data.

However Puppetry stores the following data on user PC:

  • test project created by the user (JSON files in a folder chosen by the user)
  • settings in Electron's localStorage:
    • on Linux: ~/.config/puppetry
    • on macOS: ~/Library/Logs/puppetry
    • on Windows: %USERPROFILE%\AppData\Roaming\puppetry

When using Puppetry recoder the app acts as a browser. So the user can navigate any web site. Puppetry runs the recorder in private mode, meaning no data persists between user sessions.

During test run Puppetry uses Puppeteer / Chromium to navigate URLs specified in user tests. Nonetheless Puppetry performs tests in Incognito context.

The app does not use third party services that may collect information used to identify you.

Log Data

I want to inform you that whenever you use my Service, in a case of an error in the app I collect data and information in a text file that can be found at:

  • on Windows: %USERPROFILE%\AppData\Roaming\puppetry/log.log
